添加HTTP服务监控

作为使用最多的一种提供服务方式，HTTP/HTTPS服务的监控也是很多客户需要关注的监控重点。

添加HTTP服务监控

在本系统中，HTTP服务是一种特殊类型的服务监控，所以我们选择添加一个应用服务监控

点击“应用服务监控”后，会出现一个可选服务监控插件列表，在顶部搜索区域输入HTTP后即可看到用于监控HTTP服务的插件：

点击”HTTP Checker”，即可进入HTTP 服务监控添加页面：

  1. 选择能访问到目标主机的任意采集器即可
  2. 填入必要参数，此处应输入想监控HTTP 服务的对应URL
  3. 填写上实例名，加上需要的标签，即可完成该系统的添加
  4. 稍等一会，即可看到该系统的数据
